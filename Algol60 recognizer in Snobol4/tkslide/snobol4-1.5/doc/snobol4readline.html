NAME

snobol4readline − SNOBOL4 line editing interface

SYNOPSIS

-INCLUDE ’readline.sno’

INPUT_STRING = READLINE(PROMPT_STRING)

ADD_HISTORY(STRING)

STRING = HISTORY_EXPAND(STRING)

DESCRIPTION

READLINE() performs user input with line editing.

ADD_HISTORY() adds lines to the input history available via editing commands.

SEE ALSO

snobol4(1), readline(3)

AUTHOR

Philip L. Budne
